Development tools

Python

We use Python as the main programming language for the project, it's help us setting up the RaspberryPi and the backend server. Also we use Python for the facial regconition part of the project

Flask

With Flask API, we were able to set up the connection between backend data and the HTML page via API. We then use Flask to host a Webserver by the RaspberryPi within the local network for accessing the data.

Git & GitHub

Git is a tools for control version, we use Git to manage the source code of the project and make it easier to develop the project in a team.

MySQL

We use mySQL as the database for the project, it's help us to store the data of the users and the attendance history, also the image of each access.
